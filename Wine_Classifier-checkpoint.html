Introduction

In this project we intend to use a dataset obtained from a thorough chemical examination of these wines in order to use machine learning to identify wines from various varietals grown in the same region of Italy.

There are 178 examples in the collection, each of which represents a unique wine sample with 13 chemical components, such as alcohol percentage, acidity, phenolic compounds, and color intensity. The wine cultivar, which is divided into three different groups, is the target variable for this classification task.

We will use Random Forest, k-Nearest Neighbors (k-NN), Support Vector Machines (SVM), and Logistic Regression to test the dataset. We want to determine the best method for precisely forecasting the wine cultivar based on its chemical characteristics by creating a baseline model and contrasting the results of various algorithms.

Model Training, Evaluation, and Results Compilation

In [201]:
for name, model in models.items():
    model.fit(X_train, y_train)
    y_pred = model.predict(X_test)
    y_score = model.predict_proba(X_test)

    acc = accuracy_score(y_test, y_pred)
    f1 = f1_score(y_test, y_pred, average='weighted')
    cm = confusion_matrix(y_test, y_pred)

    # Binarize labels for ROC AUC
    y_bin = label_binarize(y_test, classes=[1, 2, 3])
    auc_score = roc_auc_score(y_bin, y_score, multi_class='ovr')

    results[name] = {
        'Accuracy': acc,
        'F1 Score': f1,
        'Confusion Matrix': cm,
        'ROC AUC': auc_score,
        'y_score': y_score,
        'y_bin': y_bin
    }

   # Save for bar plots
    model_names.append(name)
    accuracies.append(acc)
    f1_scores.append(f1)
    roc_aucs.append(auc_score)

# Plot ROC curves
colors = ['red', 'green', 'blue']
classes = [1, 2, 3]

for name, metrics in results.items():
    y_bin = metrics['y_bin']
    y_score = metrics['y_score']

    fpr = dict()
    tpr = dict()
    roc_auc = dict()

    for i in range(3):  # 3 classes
        fpr[i], tpr[i], _ = roc_curve(y_bin[:, i], y_score[:, i])
        roc_auc[i] = auc(fpr[i], tpr[i])

    # Plot all ROC curves for this model
    plt.figure(figsize=(8, 6))
    for i in range(3):
        plt.plot(fpr[i], tpr[i], color=colors[i],
                 label=f'Class {classes[i]} (AUC = {roc_auc[i]:.2f})')

    plt.plot([0, 1], [0, 1], 'k--', lw=1)
    plt.xlim([0.0, 1.0])
    plt.ylim([0.0, 1.05])
    plt.title(f'ROC Curves - {name}')
    plt.xlabel('False Positive Rate')
    plt.ylabel('True Positive Rate')
    plt.legend(loc='lower right')
    plt.grid(True)
    plt.tight_layout()
    plt.show()
